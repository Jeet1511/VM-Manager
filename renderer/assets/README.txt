Place your branding images here:

- vm-xposed-logo.png   (main logo with text)
- vm-xposed-mark.png   (square icon/app mark)

The app is configured to use these automatically.
If these files are missing, it falls back to built-in SVG branding.

Also accepted automatically:
- vm-xposed-mark.ico / .webp / .jpg / .jpeg
- vm-xposed-logo.ico / .webp / .jpg / .jpeg
- icon.ico / icon.png / logo.png (and .webp/.jpg/.jpeg variants)
