Place text patch notes in this folder.

Required naming for auto-update detection:
- patch-vX.Y.Z.txt  (or .md)

Examples:
- patch-v1.0.1.txt
- patch-v1.0.2.md

Each version should have:
1. One installer in Installer\ (same X.Y.Z)
2. One patch note file here (same X.Y.Z)

The app reads the matching patch note and shows it in Settings > Updates.
