Place text patch notes in this folder.

Recommended naming for version mapping:
- patch-vX.Y.Z.txt  (or .md)

Examples:
- patch-v1.0.1.txt
- patch-v1.1.4.txt

For best update matching:
1. One installer in Installer\ (same X.Y.Z)
2. One patch note file here (same X.Y.Z)

The app reads the best matching patch note and shows it in Settings > Updates.
